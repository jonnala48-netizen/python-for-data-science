## 📋 About This Course

This course is designed to equip learners with the Python programming skills needed to pursue a career in **Data Science**. The curriculum follows a structured, project-driven approach — starting from core Python fundamentals and progressively building towards advanced analytics, machine learning, and professional practice.

The programme is split into **12 modules** across **4 phases**, totalling approximately **22.5 weeks** of structured learning. Each module includes a study guide, practical Jupyter notebook exercises, and hands-on mini-projects to reinforce concepts.

### 🎯 Who Is This For?

- Aspiring data scientists and analysts
- Software developers transitioning into data roles
- Professionals looking to add Python-based data skills to their toolkit
- **No prior programming experience required** — we start from the very basics

### ✅ What You Will Learn

Upon completion, you will be able to:

- Write clean, efficient Python code following professional best practices
- Manipulate, clean, and analyse large datasets using NumPy and Pandas
- Create compelling data visualisations for exploratory and explanatory purposes
- Apply statistical methods and hypothesis testing to real-world problems
- Build, evaluate, and tune machine learning models using scikit-learn
- Conduct end-to-end data science projects from data ingestion to model deployment
- Present findings effectively through Jupyter Notebooks and visual storytelling

## 🚀 Getting Started

> ### 🎓 New Learner?
> **Start here → [GETTING_STARTED.md](GETTING_STARTED.md)** — Step-by-step guide to fork this repo, set up your environment, and open your first notebook.
>
> **Never used Git?** → [GIT_CHEAT_SHEET.md](GIT_CHEAT_SHEET.md) — All the commands you need on one page.

### Prerequisites

- A computer (Windows, macOS, or Linux)
- Basic computer literacy
- A willingness to learn!

### Option 1: Local Setup (Recommended)

**Step 1 — Install Python**

Download and install Python from [python.org/downloads](https://www.python.org/downloads/). On Windows, **tick "Add python.exe to PATH"** during installation.

**Step 2 — Install VS Code**

Download [VS Code](https://code.visualstudio.com/) and install the **Python** and **Jupyter** extensions from the Extensions marketplace.

**Step 3 — Clone this repository**

```bash
git clone https://github.com/YOUR-USERNAME/python-for-data-science.git
cd python-for-data-science
```

**Step 4 — Create a virtual environment and install dependencies**

```powershell
# Windows (PowerShell)
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

```bash
# macOS / Linux
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

**Step 5 — Open in VS Code**

```bash
code .
```

Open any `.ipynb` file → select the `.venv` Python interpreter → run cells with `Shift + Enter`.

### Option 2: Google Colab (No Installation)

1. Go to [Google Colab](https://colab.research.google.com/)
2. Click **File → Open notebook → GitHub**
3. Paste this repository URL
4. Select any notebook to open and run it in the cloud

## 📐 How to Use This Course

1. **Follow the modules in order** — each builds on the previous one
2. **Read the markdown cells** before running code — they explain the concepts
3. **Run every code cell** — press `Shift + Enter` to execute
4. **Do the exercises** — they range from ⭐ (beginner) to ⭐⭐⭐ (challenging)
5. **Check the solutions** only after attempting the exercises yourself
6. **Experiment!** — Change values, break things, and see what happens

### 💡 Tips for Success

- **Consistency beats intensity** — 1 hour daily is better than 7 hours once a week
- **Type the code yourself** — don't just read it or copy-paste
- **Errors are your friends** — every error message teaches you something
- **Use comments** — annotate your code to reinforce understanding
